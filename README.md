# TestOne
Aministrador de Tareas con C#.
Es un programa creado en C#, para monitorear los recursos de una computadora,
como el consumo del Procesador, la memoria, el o los discos duros, conexiones Wifi o Lan.

Usando galerias como.:
Net.NetworkInformation, System.Diagnostics y VisualBasic.Devices
Controles utilizados
* DataVisualization.Charting.Chart
* Listbox, progressBar
Creado con .NETCore.App
